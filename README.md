# KeyboardDemo
   前段时间做了一个需求，类似验证码输入框，但输入的优惠码有数字和大小写字母，所以就需要用到自定义软键盘，不然总是切换数字与字母太麻烦，用户体验不佳。
刚开始想着到网上找一些demo得了，可iOS倒是有很多，安卓却没找到，索性自己撸一个。
